	<b>Степа́н Андрі́йович Банде́ра</b><a href="#">(1 січня</a></b> <a href="#">1909,</a>с. <a href="#">Старий Угринів</a>, <a href="#">Калуський повіт</a>,<a href="#">Королівство Галичини та Володимирії</a>, <a href="#">Австро-Угорщина</a>, нині <a href="#">Калуського району</a>, <a href="#">Івано-Франківська область</a>, <a href="#">Україна</a> — <a href="#">15 жовтня</a> <a href="#">1959</a>, <a href="#">Мюнхен</a>, <a href="#>"Баварія</a>, <a href="#">Федеративна Республіка Німеччини</a>) — <a href="#">український</a> <a href="#">політичний діяч</a>, революціонер, один із радикальних та чільних ідеологів, практиків і теоретиків українського націоналістичного руху <a href="#">XX століття</a>, після розколу <a href="#">Організації українських націоналістів</a> — голова Проводу <a href="#">ОУН-Б[2]</a>. Разом з <a href="#">Миколою Лебедем</a> був одним зі організаторів <a href="#">вбивства Броніслава Перацького</a> за політику <a href="#">пацифікації</a> та <a href="#">вбивства Олексія Майлова</a> через провед       	спланованого та штучного організованого <a href="#">голоду на українських землях 1932—1933</a>.
		
	<p>Як активний діяч <a href="#">УВО</a> та <a href="#">ОУН</a> має законодавчо визначений статус <a href="#">Борець за незалежність України у XX сторіччі</a>. В 2010 посмертно нагороджений званням <a href="#">герой України</a>. Вважається <a href="#">національним героєм</a> України. У 2008 році був визнаний третім зі ста великих українців після <a href="#">Ярослава Мудрого</a> та <a href="#">Миколи Амосова</a> за підсумками голосів в опитуванні громадської думи <a href="#">"Великі Українці"</a>
	<p>Степан Бандера та <a href="#">Ярослав Стецько</a> були авторами <a href="#">Акта відновлення Української Держави</a> <a href="#">30 червня</a> <a href="#">1941</a> року. <a href="#">5 липня</a> <a href="#">1941</a> року Бандеру помістили під домашній арешт, а з <a href="#">15 вересня</a> <a href="#">1941</a> року — у центральну Берлінську в'язницю. З початку <a href="#">1942</a> року по серпень <a href="#">1944</a> року перебував у концтаборі <a href="#">Заксенгаузен</a> у бункері <a href="#">«Целленбау»</a> у відносно комфортних умовах[4]. У вересні 1944 року його звільнили й запропонували участь у керівництві антирадянського збройного руху в тилу <a href="#">Червоної армії[5][6]</a>, однак Бандера відхилив пропозицію та на співпрацю не погодився[7][8]. С. Бандера і <a href="#">Ярослав Стецько</a> у квітні 1944 року на нараді обговорювали завдання підривної діяльності проти СРСР з керівником таємних операцій вермахту <a href="#">Отто Скорцені.[9][10]</a><a href="#"> Представники ОУН(б) та УПА, яку створила ОУН(б), в цей час проводили переговори, почали і продовжували співпрацювати з німцями,[11] [12] [13] [14] [15] [16] [17] хоча інші підрозділи УПА ще воювали з нацистами в 1944 році.[18] Разом з <a href="#">Андрієм Мельником</a>, <a href="#">Адрієм Лівицьким</a> та <a href="#">Павлом Скоропадським</a> став співзасновником <a href="#">Українського національного комітету</a>.
